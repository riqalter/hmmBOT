## How to config ?

```json

{
  "prefix": "your prefix",
  "token": "your token",
  "owners": ["OWNER ID"],
  "footer": "NSFW Bot",

  "msg": {
    "nsfwWarn": "You must use this command in an nsfw channel!",
    "loading": "Please wait...",
    "imageNotLoading": "Image not loading ? Click Here"
  }
}
```
